Intro to Java

  Data Types:
  int     - Whole number
  boolean - True/False
  char    - Single character
  
  Variables     - Used to store values
  Whitespace    - Makes code readable, java ignores it
  Comments (//) - Describe code and its purpose
  
  Arithmetic Operators:
  +, -, *, /, %
  
  Relational Operators:
  <, <=, >, >=
  
  Equality Operators:
  ==, !=

Conditionals and Control Flow

Order of Boolean Operations:
1. ! (NOT)
2. && (AND)
3. || (OR)
